# Changelog

Все значимые изменения базы знаний SWC фиксируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
версионирование - [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-04-22

### Добавлено

- **core/** - стабильное ядро из 8 документов: ecosystem, swc-pass, swc-field, road-to-1m, partner-program, brand-voice, visual-style, legal-disclaimers
- **projects/** - описания проектов: atlas, smart, cryptadium + INDEX.md
- **faq/** - живая база FAQ: general, swc-pass, swc-field, road-to-1m, atlas + INDEX.md
- **updates/** - папка для официальных объявлений (пока пустая)
- README.md с описанием структуры и правил ведения
- VERSION с текущей версией

### Исходник

База знаний собрана на основе документа «ИИ-Ассистент Сотрудника SWC v1.0» (2025).

## Принципы версионирования

- **Major (X.0.0)** - значительные изменения структуры, несовместимые правки в бренд-гайде
- **Minor (1.X.0)** - новые разделы, новые проекты на витрине, новые темы FAQ
- **Patch (1.0.X)** - уточнения формулировок, исправления опечаток, мелкие правки
