# Cryptadium - криптовалютная платёжная система

## Что это

**Cryptadium** - криптовалютная платёжная система для бизнеса (B2B). Позволяет принимать оплату в криптовалюте легально, быстро и безопасно.

## Юридические параметры

- **Лицензия:** ЕС
- **Юридическое лицо:** CE-International UAB
- **Местоположение:** Вильнюс, Литва

## Целевая аудитория (для кого)

- E-commerce
- SaaS
- Разработка ПО
- Gambling
- Betting
- Forex
- I-gaming

## Ключевые преимущества для B2B

| Параметр | Значение |
|---|---|
| Комиссия | 0,7-0,9% |
| Время интеграции | 1 день |
| KYB | Несколько часов |
| Поддерживаемые валюты | Самые популярные криптовалюты |
| Защита от волатильности | Конвертация в стейблкоины |
| Вывод средств | Бесплатный, без ограничений |
| Время работы | Мгновенные глобальные платежи 24/7 |

## Партнёрская программа Cryptadium

- **Пожизненные выплаты 30% комиссии** от магазинов, подключённых партнёром
- **Цель для партнёра:** 5+ новых интернет-магазинов в месяц

## Приоритет в стратегии SWC

Cryptadium - **параллельный поток** в стратегии SWC. Не должен отнимать ресурсы у Atlas (Приоритет №1).
