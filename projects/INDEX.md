## Как этот индекс используется

Навык SWC-assistant читает этот файл, чтобы узнать актуальный список проектов. При вопросе о конкретном проекте навык дополнительно загружает соответствующий файл из `projects/`.

## Как добавить новый проект

1. Создать файл `projects/название-проекта.md` по шаблону
2. Добавить строку в таблицу выше
3. Закоммитить в `main` - навык увидит изменение при следующем запросе
