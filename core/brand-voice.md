# 08. Бренд-гайд - формулировки и терминология

**Это самый критический файл для коммуникации.** Любой текст о SWC должен быть проверен против правил этого раздела перед отправкой.

## Допустимые названия и позиционирование

### Допустимые названия

- Sky World Community
- SWC
- Экосистема SWC
- Мультипроектная экосистема для партнёров

### Запрещённые названия / позиционирование

- Любые сторонние бренды (UST, GTI и т.д. - как часть позиционирования SWC)
- «инвестиционная компания»
- «финансовая платформа»

### SWC - это:

- Экосистема
- Платформа
- Среда взаимодействия
- Инструмент для работы с проектами

## Базовая логика построения любого текста

**Правильная логика:**
```
контекст (экосистема) → инструмент (платформа / сервис) → участие → вознаграждение
```

**Неверная логика (запрещена):**
```
инвестиции → доход → прибыль → структура → заработок
```

## Готовые формулировки

### Про продукт
> «SWC - это мультипроектная экосистема, которая объединяет проекты, сервисы и партнёрское сообщество.»

### Про участие
> «Участники экосистемы получают доступ к инструментам платформы и могут выстраивать собственную стратегию работы с активами.»

### Про ценность
> «Это позволяет действовать системно и принимать решения на основе структурированной информации с учётом самостоятельной проверки материалов.»

### Про партнёрскую программу
> «В рамках партнёрской программы предусмотрено агентское вознаграждение.»

## Стиль речи

**Общий тон:** спокойный, структурный, без давления, без обещаний.

| Избегаем | Используем |
|---|---|
| «лучший» | «позволяет выстроить подход» |
| «уникальный шанс» | «даёт возможность» |
| «заработаете» | «формирует понимание» |
| «не упустите» | «помогает структурировать» |

## Категории запрещённой / допустимой терминологии

### 1. Деньги и вознаграждение

**Запрещено:**
- заработок
- доход
- прибыль
- пассивный доход
- гарантии

**Используем:**
- агентское вознаграждение
- возможность участия
- результаты партнёрского сообщества
- экономическая ценность

**Пример:**
- ❌ «вы можете зарабатывать»
- ✅ «вы можете получать агентское вознаграждение»

### 2. Инвестиции

**Запрещено:**
- инвестировать
- вложить деньги
- инвестор
- инвестиционный портфель

**Используем:**
- участвовать в проектах
- стратегия работы с активами
- участник платформы
- проекты экосистемы

**Пример:**
- ❌ «инвестировать в проект»
- ✅ «участвовать в проекте платформы»

### 3. Партнёрская модель

**Запрещено:**
- MLM
- структура
- рефералы
- уровни

**Используем:**
- партнёрское сообщество
- партнёрская программа
- команда партнёров
- участники

**Пример:**
- ❌ «развивать структуру»
- ✅ «развивать партнёрское сообщество»

### 4. Продажи и продвижение

**Запрещено:**
- продавать
- зарабатывать на людях
- привлекать деньги
- привлекать партнёров

**Используем:**
- представлять возможности платформы
- расширять партнёрское сообщество
- масштабировать сообщество

### 5. Финансовые обещания - СТРОГО ЗАПРЕЩЕНО

**Несёт юридическую ответственность! Запрещено:**
- гарантированный доход
- окупаемость
- доходность
- без рисков
- гарантированная прибыль
- многоуровневый маркетинг
- заработок без вложений
- отсутствие рисков

**Используем:**
- «может создавать экономическую ценность»

## Финальный принцип (главный тест любого текста)

> Любой текст SWC должен отвечать на вопрос **«Как человек участвует в экосистеме?»** - а не **«Сколько он заработает?»**

## Чек-лист самопроверки перед отправкой любого текста

- [ ] Нет слов: заработок, доход, прибыль, инвестор, MLM, структура, рефералы, гарантия
- [ ] Нет обещаний доходности или отсутствия рисков
- [ ] Сохранена логика «экосистема → инструмент → участие → вознаграждение»
- [ ] Использованы корректные названия продуктов (SWC Pass / SWC Field / Road to $1M / Atlas)
- [ ] Тон спокойный и структурный, без давления
- [ ] Если затрагиваются финансовые результаты - есть дисклеймер (см. файл 11-legal-disclaimers.md)
