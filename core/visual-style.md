# 09. Визуальный стиль - цвета, шрифты, оформление документов

## Процесс работы с задачами

### Этап 1 - работа в чате

Сначала задача прорабатывается в режиме переписки. Идёт обсуждение, уточнение, структурирование. Текст, логика, содержание - всё доводится до нужного результата прямо в диалоге.

**Не нужно торопиться делать «финальный документ»** до тех пор, пока содержание не готово.

### Этап 2 - финальный документ

Когда задача доведена до конца и результат устраивает, нужно спросить: **в каком формате делаем итоговый документ?**

Варианты форматов:
- **HTML** - для публикации, лендингов, онлайн-материалов
- **PDF** - для официальных материалов, отправки клиентам и партнёрам
- **.docx** - для документов, которые будут редактироваться
- **Google Docs** - для совместной работы
- **Markdown** - для технической документации и базы знаний
- **.pptx** - для выступлений и питчей

**Правила:**
- Нельзя делать финальный документ, не выяснив формат
- Нельзя уточнять формат, пока содержание не готово

## Цветовые системы продуктов

SWC - экосистема нескольких направлений. **Каждое направление имеет свой визуальный язык.**

Правила:
- Нельзя использовать цвета одного продукта в материалах другого
- Нельзя смешивать палитры
- Все цвета ниже - **единственные допустимые**

## Типографика

### Основной шрифт: Muller Next Wide / Muller Next

| Элемент | Шрифт |
|---|---|
| H1 | Muller Next Wide Bold |
| H2 | Muller Next Wide Medium |
| Подзаголовки | Muller Next Bold |
| Основной текст | Muller Next Regular |

### Запасной (если недоступен Muller)

- Inter Bold / Inter Medium / Inter Regular

### Запрещено

- Засечки (serif-шрифты)
- Курсив
- Обводка
- Начертание Light

## Стиль оформления документов

**Последовательность работы:**

1. **Определить направление документа** - и применять только его цветовую систему. Смешивать палитры разных направлений нельзя.

2. **Заголовки разделов:**
   - Акцентный цвет направления
   - Bold
   - Без точки в конце

3. **Таблицы:**
   - Заголовки: тёмный фон `#1C2128` + белый текст
   - Строки: чередование White `#FFFFFF` и Anti-flash White `#EBEBEB`

4. **Логотип SWC:**
   - В левом верхнем **или** нижнем углу
   - На всех официальных документах

5. **Запрещено:**
   - Градиенты в рабочих документах
   - Декоративные элементы вне утверждённой палитры SWC
