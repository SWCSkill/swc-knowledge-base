# Индекс FAQ экосистемы SWC

Живая база ответов на типовые вопросы партнёров. Обновляется по мере появления новых вопросов и уточнения формулировок.

## Доступные разделы FAQ

| Тема | Файл |
|---|---|
| Общие вопросы, трансформация SWC, экосистема | `faq/general.md` |
| SWC Pass - подписка, тарифы | `faq/swc-pass.md` |
| SWC Field - витрина, проекты | `faq/swc-field.md` |
| Road to $1M - стратегия, аналитика | `faq/road-to-1m.md` |
| Atlas - параметры, риски | `faq/atlas.md` |

## Как искать ответ

1. Определи тему вопроса партнёра
2. Загрузи соответствующий файл
3. Если ответа там нет - посмотри в `core/` и `projects/`
4. Если вопрос юридически чувствительный - **обязательно** сверься с `core/legal-disclaimers.md` и `core/brand-voice.md`

## Как добавить новый вопрос

1. Открыть нужный файл `faq/*.md`
2. Добавить блок вопроса в формате `### Вопрос` + ответ ниже
3. Закоммитить в `main`

## Принципы ответов

- Всегда используй допустимую терминологию (`core/brand-voice.md`)
- Никогда не обещай доходность
- При упоминании цифр - добавляй дисклеймер
- Логика: экосистема → инструмент → участие → вознаграждение
