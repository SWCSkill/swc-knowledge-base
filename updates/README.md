# Updates

Папка для официальных объявлений компании SWC.

## Формат файла

```
updates/2026-Q1-updates.md
updates/2026-04-15-new-tariff.md
updates/2026-04-20-atlas-milestone.md
```

Название файла: `YYYY-MM-DD-краткое-описание.md` или `YYYY-Qn-summary.md`.

## Что сюда класть

- Изменения тарифов SWC Pass
- Новые проекты на витрине (анонсы)
- Обновления партнёрской программы
- Важные стратегические объявления
- Изменения в партнёрском вознаграждении

## Как навык использует эту папку

Навык проверяет последние 2-3 файла из этой папки при общих вопросах о статусе компании, новостях, недавних изменениях.
